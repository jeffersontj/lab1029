How I Update This Page / 我如何更新這個頁面

  1. 確認目前在D:\work\lab1029 分支： 
                            D:
                        cd \work
  2. 儲存檔案後，回到PowerShell 或 Terminal，確認目前在 D:\work\lab1029 分支： 
                            git commit -a -m "My 8th commit."
    版本名稱可隨意輸入。
  3. 若想跟前幾個版本做對比，可以輸入： 
                            git show -1 #與前一個版本對比
    或者： 
                            git show -2 #與前兩個版本對比
    若想退出對比畫面，請按[:q]。 (VI編輯器指令)
  4. 若恢復至前幾個版本，可以先輸入： 
                            git log --pretty=oneline
    找到想恢復的版本編號（例如：3ee10825adc2cf23a35e27fbe3ca3a9473636a83）
    然後輸入： 
                            git restore --source 3ee10825adc2cf23a35e27fbe3ca3a9473636a83 resume-M11405505.html
  5. 把更新推回 GitHub 遠端： 
                            git push origin feature/M11405505
  6. 到 GitHub 建立或更新 Pull Request
  7. 複製 GitHub 的更新置本機端覆蓋 
                            git clone https://github.com/jeffersontj/lab1029.git
